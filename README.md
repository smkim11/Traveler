# Traveler
amadeus API를 통해 원하는 여행지의 항공권을 검색하고, AI chatbot을 통해 가장 저렴한 항공권을 물어보거나 관광지에 대해 질문해 답변을 얻을 수 있는 사이트 입니다.
# 팀원
- 김성민: UI, chat bot API 연동, 로그인
- 이민제: chat bot API 연동, 로그인, 기록 저장 기능
- 유시연: amadeus API 연동, 각 기능 연계
- 노일권: UI
